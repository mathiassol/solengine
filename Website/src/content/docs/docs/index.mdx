# Introduction

SolEngine is a high-performance, minimal engine built for modern workflows.
This documentation covers everything from installation to advanced usage.

## What is SolEngine?

SolEngine provides a lean foundation for building fast, efficient applications
without the overhead of bloated frameworks. It is modular by design — use only
what you need.

## Key Features

- **Fast** — Optimized core with minimal overhead
- **Modular** — Pick the components that suit your project
- **Cross-platform** — Windows, macOS, and Linux
- **Zero dependencies** — Self-contained and easy to distribute
